Web conversión grados php
Realiza una web de conversión de grados de celsius a farenheit.
La página inicial tendrá un cuadro de texto, en el que el usuario podrá  introducir un numero.
Al lado habrá dos botones, uno para convertir a celsius y otro para convertir a farenheit.
 

La aplicación tendrá que manejar con bloques try..catch los siguientes errores:
 

Control que se introduzca un valor numerico
los campos tienen que tener datos.
Poner limites inferiores y superiores razonablemente con los tipos de cantidades a convertir.
Se mostrará al apretar uno de los botones el resultado de la conversón.
Pasarme el link al repositorio con el código en GitHub
